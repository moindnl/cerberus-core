# TireIQ

Tire pressure calculator for cyclists.

## Features

- **Pressure Calculation** - Calculates optimal tire pressure based on tire model, rider weight, surface type, and tubeless setup
- **Unit Toggle** - Switch between bar and psi
- **Tire Models** - Pre-configured models for Road, Touring, Gravel, and MTB tires
- **Surface Types** - Adjusts for Road, Wet, Race track, Gravel, Cobblestone, Forest, and Mud
- **Tubeless Support** - Automatic pressure adjustment for tubeless tires
- **Wheel Position** - Different base pressures for front and rear wheels
- **Weight Factor** - Adjusts pressure based on rider weight (40-200kg)
- **Temperature Tips** - Recommendations for cold and hot conditions
- **Fully Responsive** - Works on desktop and mobile devices
- **Mobile Nav Bar** - Floating bottom navigation with liquid glass effect (scroll to top, toggle unit, share)

## Install & Run

```bash
# Clone the repository
git clone https://github.com/moindnl/cerberus-core.git
cd cerberus-core

# Install dependencies
npm install

# Run on localhost
npm run dev
```

Run it on localhost or host it on your own server. I don't care. I build this stuff for fun. Ride on!
